# Rasa1.8_Weather_Chatbot
## create virtual environment python3.6
```
python3 -m venv venv
```
## activate virtual environment
```
source venv/bin/activate
```
## Install rasa1.8
```
pip install rasa==1.8
```
## create initial bot
```
rasa init --no-prompt
```
## Replace all file 
## For trainig
```
rasa train
```
## For test
```
rasa run actions
```
##open another tab
```
rasa shell
```
